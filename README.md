TYPING SPEED CALCULATOR
A simple Python-based **Typing Speed Calculator** that helps users measure their typing performance. The application calculates **Words Per Minute (WPM)** and **Accuracy** by comparing the user's typed text with a randomly selected paragraph.

FEATURES :
~ Random paragraph selection from a JSON file
~ Calculates typing speed (Words Per Minute)
~ Calculates typing accuracy
~ Calculates total time taken
~ Uses JSON files for storing typing paragraphs

TECHNOLOGIES USED
~ Python 3
~ JSON
~ Time Module
